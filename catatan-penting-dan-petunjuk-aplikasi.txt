# System aplikasi : 
	1.Menggunakan standar ES6 dalam menuliskan kode JavaScript
	2.Terdapat minimal satu penerapan custom element
	3.Menggunakan Webpack sebagai module bundler dan environment development
	4.kan API dengan menggunakan konsep AJAX
# Cara memakai aplikasi film :
	1.Simpan project di web server seperti apache yang umumnya memakai xampp di folder htdocs ( agar menghindari error di block oleh CORS )
	2.Masuk ke path project film di terminal,misal -> C:\xampp\htocs\film
	3.Install plugin yang di butuhkan dengan mengetikan perintah " npm install "
	4.Jalankan aplikasi dengan mengetikan perintah " npm run start-dev " di terminal
	5.Buka url di browser dengan alamat http://localhost:8080/
	6.Nikmati aplikasi dengan pasilitas mencari film kesukaan kita :D , Kalau saya suka film shooter / film genre perang sniper
